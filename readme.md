# Forum App

A modern, user-friendly **Forum App** that allows users to create, discuss, and interact in topic-based discussions. Built for learning, collaboration, and community engagement.

- [Tech Stack](#tech-stack)

---

## Features
- User registration and authentication  
- Create, edit, and delete posts and threads  
- Comment and reply to posts  
- Upvote and downvote functionality  
- Search and filter threads by topics  
- Responsive UI for desktop and mobile  

---

## Tech Stack
- **Frontend:** React.js  
- **Backend:** Node.js + Express  
- **Database:** MongoDB  
- **Authentication:** JWT  
- **Styling:** CSS / Tailwind

## Getting Started

### Prerequisites
- Node.js v14 or higher  
- npm or yarn  
- MongoDB  

### Installation
1. Clone the repository:  
```bash
git clone https://github.com/Ayyappan-Subramanian/Forum-Project-DevSecOps.git
cd forum-app
```

2. Install dependencies:
```bash
npm install
```
3. Configure environment variables:
```bash
PORT=5000
DB_URL=your_database_connection_string
JWT_SECRET=your_jwt_secret
```
4. Start the backend server:
```bash
npm run server
```
Start the frontend:
```bash
npm start
```

The app should now be running at http://localhost:3000.

### Usage

- Register a new user account.

- Create a new thread or browse existing threads.

- Comment, reply, and vote on posts.

- Explore the search and filtering options to find discussions of interest.
